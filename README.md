# lullaby
Idol website event calender crawler backend made with .NET Core MVC + C#.

## Supported format for calender
- iCal
- JSON

## Supported idol groups with this application
- 群青の世界
- Kolokol
- On the treat Super Season
- 手羽先センセーション
- yosugala
- Axelight
- PRSMIN
etc...
